# 🛒 GreenCart

GreenCart is a full-stack e-commerce web application built using the MERN stack. It includes JWT authentication, product management, cart functionality, and secure Stripe payments.

## 🚀 Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js
* JWT
* Stripe
* Cloudinary

## ✨ Features

* User Login & Register
* Browse Products
* Add to Cart
* Online Payment
* Admin Product Control

## ▶️ Run Project

```bash
npm install
npm run dev
```
